# Happiee Shoppiee 🐾

A cute E-commerce prototype built with **React**, **CSS**, and **Java Spring Boot**.

## Features
- Soft UI Design based on Figma Prototypes.
- Product Catalog for Plushies and Stationery.
- Full Checkout Flow with Shipping Information.

## Tech Stack
- **Frontend:** React.js, React Router
- **Backend:** Java Spring Boot
- **Styling:** Custom CSS3 (Flexbox & Grid)

## How to Run
1. Clone the repo: `git clone [your-repo-link]`
2. Start Frontend: `npm install && npm start`
3. Start Backend: Run `HappieeShoppieeApplication.java` in your IDE.
